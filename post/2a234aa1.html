VsCode配置Vue模板

前端 Vue VS Code Vue 工具 2019/06/20

VS Code拥有很多特别适合和方便的功能，自定义文件模板就是其中之一，本文中主要记录Vue的模板

创建模板步骤

  • 点击VS Code左下角设置按钮，选择用户代码片段
  • 选择新建全局代码片段，输入名称，例如vue
  • 编辑文件即可
